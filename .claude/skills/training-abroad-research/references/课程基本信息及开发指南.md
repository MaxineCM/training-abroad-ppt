# 方案综述

本方案定位为一套可复制、可适配的通用课程框架，适用于为中国企业外派至全球任一目标国家的员工提供行前培训。
目标国通用商务规则模块，系统涵盖国家商务运营规则与主要商务风险，帮助外派员工在抵达前建立清晰的商业环境认知，从而在第一时间做到守规合规、识险避险，大幅降低因规则不熟悉导致的经营风险。

## 核心设计原则：

- 模板统一：所有国家课程遵循相同的大纲结构与章节逻辑
- 内容分层：通用内容（固定）+ 国别内容（可替换）+ 企业定制内容（留白）
- 适配灵活：支持短期出差与长期常驻两类受众
- 生产高效：便于批量开发不同国家的版本
- 风险前置：在履职技能之前系统学习目标国商务规则与风险，实现"知规—知险—才能知行"

## 一、课程基本信息

| 项目 | 内容 |
|------|------|
| 课程名称 | 《扬帆海外：[目标国名]从零到一的外派快速适应指南》 |
| 课程类型 | 行前培训 / 跨文化适应 / 商务合规 / 安全常驻 |
| 适用对象 | 短期出差（≤3个月）与长期常驻（≥6个月）的中方员工 |
| 标准时长 | 3小时（V2.0新增商务规则模块约30分钟） |
| 章节结构 | 6章 + 3个附录 |
| 适配方式 | 通用内容固定 + 国别信息替换 + 企业信息留白 |
| 版本说明 | V2.0：新增第5章"目标国通用商务规则"，原第5章顺延为第6章 |

## 二、课程目标（通用版）

1. 建立全景认知：掌握目标国政治、经济、社会、文化及双边关系的核心信息，构建系统性认知框架。
2. 完成行前准备：熟悉签证、保险、健康、行李及心态建设等关键事务的办理流程。
3. 掌握安顿与安全技能：获取异国生活安顿的实用方法，系统认知安全环境与风险应对法则。
4. 精进跨文化能力：理解文化禁忌与社交礼仪，掌握商务沟通与跨文化工作冲突处理技巧。
5. 掌握目标国商务规则与风险：了解目标国商务运营的核心规则框架（劳工、合规、合同、税务、知识产权等），识别并评估主要商务风险类型，在实际经营前建立清晰的规则边界意识，做到守规合规、识险避险。
6. 提升履职效率：了解与本地团队协作的方法，识别关键利益相关方与基本经营风险。

# 课程开发指南

## 一、国别信息替换规范

每个国家版本需要替换以下信息模块：

| 章节 | 需替换的模块 | 数据来源建议 |
|------|--------------|----------------|
| 第一章 | 历史、政经、经济、宗教、社会文化、双边关系 | 外交部网站、经济学人、本地调研 |
| 2.1 | 签证类型与流程 | 目标国使领馆官网 |
| 2.2 | 疫苗接种要求 | WHO、CDC、目标国卫生部 |
| 3.2 | 住房、银行、交通、App | 本地生活指南、外派社区 |
| 4.1-4.2 | 宗教、禁忌、礼仪 | 跨文化研究、本地专家 |
| 5.1/5.3 | 国别商务法律框架与主要商务风险 | 当地法律机构、商务部国别报告、在华使馆经商处【新增】 |
| 附录C | 紧急电话、使领馆信息 | 外交部领保中心 |
