# 外派行前培训 — PPT课件生成器

## 概述

本 Skill 读取 `Output/PPT内容文件/[国家名]出国培训课程文档/` 下的章节内容文档（.docx），复用 `references/新加坡PPT样例.pptx` 的设计风格（配色、字体、布局），通过 pptx Skill 的 html2pptx 工作流逐章生成 .pptx 课件。

## 核心原则

1. **复用模板风格**：颜色、字体、布局对齐新加坡PPT样例
2. **内容不编造**：所有内容来自章节内容文档，不添加虚构信息
3. **全程自动**：一次生成全部7个 pptx 文件，不询问用户
4. **版本管理**：调整后生成新版本文件（V1→V2→V3），确认后清理旧版
5. **图文并茂**：每页幻灯片必须包含至少一张图片，禁止纯文字页面
6. **专业排版**：禁止原封不动粘贴文字和表格，必须提炼要点、信息图形化、使用图标和数据可视化

## 图片获取策略

### 官方内容：搜索真实图片（禁止 AI 生成）

以下类型必须通过 **Tavily MCP 图片搜索**获取真实图片：

| 类型 | 示例 | 获取方式 |
|------|------|----------|
| 国旗/国徽 | 日章旗、十六瓣菊纹 | Tavily 图片搜索 |
| 政治人物肖像 | 首相、天皇 | Tavily 图片搜索 |
| 官方建筑 | 国会议事堂、皇居 | Tavily 图片搜索 |
| 地图/数据图表 | 日本地图、GDP 趋势图 | Tavily 图片搜索 |
| 货币/证件 | 日元纸币、在留卡样本 | Tavily 图片搜索 |
| 官方 Logo/标志 | 政府机构 Logo | Tavily 图片搜索 |

### 非官方内容：AI 生成配图

以下类型使用 **volcengine-image-gen Skill**（火山引擎 Seedream 4.5）生成：

| 类型 | 示例 | Prompt 要求 |
|------|------|------------|
| 场景氛围图 | 商务会议、城市景观、自然风光 | 写实风格，16:9，匹配国家特色 |
| 概念插图 | 文化符号、礼仪示意、风险图标 | 扁平化插画风格 |
| 封面/过渡页背景 | 章节封面全幅图 | 大气商务风，匹配主题色 |
| 流程/结构图 | 申请流程、组织架构 | 信息图风格 |

**调用方式**：
```bash
cd .agents/skills/volcengine-image-gen
python scripts/generate.py "prompt描述" -r 16:9 -o "输出路径.png"
```

### 图片处理规范

- 每页至少 1 张图片（封面全幅背景 + 内页配图/图标）
- 图片尺寸适配 16:9（1280×720 或 1920×1080）
- 存储至 `workspace/ppt_images/` 目录，按章节命名
- 使用 `PptxGenJS` 的 `addImage` 插入

## 依赖

- `pptxgenjs`（npm 包，PptxGenJS 纯矢量设计生成，**首选方案**）
- `pptx` Skill（html2pptx 工作流，备选方案 / 需要 playwright + sharp）
- `volcengine-image-gen` Skill（火山引擎 Seedream 4.5，按需生成配图）
- Tavily MCP（官方图片搜索）
- 新加坡PPT样例模板已在 `references/` 目录（配色参考）

## 工作流

### Step 1：提取模板设计风格

解包 `references/新加坡PPT样例.pptx`，分析主题配色和字体：

- 主题色：深蓝 #44546A、蓝色 #4472C4、橙色 #ED7D31、绿色 #70AD47、金色 #FFC000
- 背景色：白色 #FFFFFF、浅灰 #E7E6E6
- 字体：Arial（标题 + 正文）
- 幻灯片尺寸：16:9

### Step 2：读取章节内容文档

读取 `Output/PPT内容文件/[国家名]出国培训课程文档/` 下全部7个文件：
- 国家名第一章课程内容.docx → 国家名第六章课程内容.docx
- 国家名附录.docx

提取每页幻灯片的标题、内容要点、表格数据、视觉建议。

### Step 3：生成PPT课件（设计级质量 + 纯矢量为主）

#### 3.1 核心策略：设计优先

**首选方案：PptxGenJS 纯矢量设计**（无需 playwright，直接生成设计级 PPT）

使用 PptxGenJS 的形状、卡片、时间轴、大数字、视觉层次等元素构建专业幻灯片。**图片仅作为点缀**，核心视觉冲击力来自排版和形状设计。

#### 3.2 配色与字体

```javascript
const C = { navy:'1a2639', blue:'2b5797', orange:'e07b39', white:'ffffff',
            lgray:'f0f2f5', gray:'f8f9fb', text:'333333', sub:'888888', bd:'e8e8e8' };
const FONT = 'Arial';
```

#### 3.3 页面类型模板

**封面页**（每章1页）：
- 深蓝全幅背景 (`1a2639`)
- 顶部橙色细条 (`e07b39`, 6px)
- 半透明装饰圆（蓝色/橙色, transparency 50-60%）
- 章节号橙色小字 + 大标题白色 44pt + 副标题灰色

**总览页**（每章1页）：
- 6宫格图标卡片（卡片：白色圆角·阴影·图标30pt·标签10pt）
- 左侧配 3-5 条要点列表（橙色圆点 + 11pt 文字）

**数据页**（多列大数字卡）：
- 大数字 42pt 深蓝加粗居中
- 单位12pt灰色 → 橙色分割线 → 标签13pt → 说明9pt
- 卡片白色背景 + 阴影

**时间轴页**：
- 水平线 + 橙色节点圆点
- 每节点：年代11pt加粗 + 标题10pt + 说明9pt

**指标页**（左指标 + 右环图）：
- 左侧：指标名/数值/趋势，灰色分割线
- 右侧：同心圆环图（大圆浅灰外圈 + 深蓝内圈 + 白色大字）

**表格页**（数据可视化）：
- 彩色表头深蓝白字 + 交替行色
- 右侧配图/图标/色块

**小结页**（深蓝全幅）：
- 深蓝背景 + 2×3 宫格卡片（半透明背景 + 橙色左边框）
- 卡片内标题白字 + 说明灰色小字

#### 3.4 图片使用规范

**仅用于绝对不能缺少图片的场景**，优先用矢量形状代替：

| 场景 | 方式 | 说明 |
|------|------|------|
| 国旗 | Tavily 搜索 + addImage | 必须真实 |
| 地图 | 矢量形状模拟或 Tavily 搜索 | 可用形状近似 |
| 场景氛围 | 火山引擎 AI 生成 | 封面背景等 |

图片存储至 `workspace/ppt_images/`，插入时控制尺寸避免文件过大。

#### 3.5 生成流程

1. 读取章节内容文档
2. 按页面类型模板设计每页结构
3. 创建 JS 脚本，使用 PptxGenJS 生成（shapes + addText + addTable）
4. 按需搜索/生成图片并插入
5. 全部 7 章一次生成

#### 3.4 输出路径
```
Output/PPT课件/[国家名]出国培训课件/
├── [国家名]第一章_国情概况.pptx
├── [国家名]第二章_签证与行前准备.pptx
├── [国家名]第三章_生活安顿与安全.pptx
├── [国家名]第四章_文化洞察与礼仪.pptx
├── [国家名]第五章_通用商务规则.pptx
├── [国家名]第六章_高效履职与团队.pptx
└── [国家名]附录_紧急联系与汇总.pptx
```

### Step 4：输出摘要

生成完成后告知用户：
- 文件夹完整路径
- 7个 pptx 文件列表及各自页数
- 提示用户审查并提出调整意见

### Step 5：响应用户调整（版本管理）

**调整原则**：只修改用户提到的部分，不擅自改其他内容。

**版本规则**：每次调整后另存为新版本文件（如 V1 → V2 → V3），不覆盖原文件。只有用户要求调整的章节生成新版本。

**最终整理**：用户确认无需调整后，询问确认后删除旧版文件，只保留最新版。
