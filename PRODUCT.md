# PRODUCT.md

## Register
**product** — A dashboard/tool for generating training-abroad PPTs. Design serves the product.

## Users & Purpose
- **Who**: Individual training program managers, working alone
- **Job**: Batch generate country-specific pre-departure training PPTs (research → content → PPT), review quality, approve or request adjustments, download final files
- **Context**: Desk work, long sessions, needs efficiency and clarity
- **Primary task per screen**:
  - Generate tab: Input country name, kick off generation pipeline, see progress
  - Review tab: Inspect generated PPT content, submit modification requests, approve
  - Completed tab: Browse finished PPTs, batch download

## Brand & Personality
- **3 words**: Professional, efficient, trustworthy
- **Anti-references**: Not playful, not consumer-app, not marketing-flashy
- **Tone**: Enterprise-calm, not startup-edgy

## Accessibility
- Desktop primary (internal tool, no mobile needed)
- Color contrast ≥ 4.5:1 for body text
